# Security Policy

## Supported Versions

| Version | Supported |
|:---|:---:|
| 0.x (current) | ✅ |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do NOT** open a public issue
2. Email: security@opsdata.ai
3. Include: description, reproduction steps, impact assessment
4. We will respond within 48 hours

## Security Design

Pylon follows OWASP Agentic Top 10 and OWASP LLM Top 10 guidelines. See the [requirements document](docs/architecture/requirements-v2.md) for full security architecture.
