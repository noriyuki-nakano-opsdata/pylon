# Setup

Use a persistent browser session for Kling AI:

```bash
agent-browser --session-name kling-ai open https://app.klingai.com/global/
```

Log in once and preserve the session for future runs.

## Practical Notes

- Prefer the global app unless the user explicitly needs the China site
- Create a run-specific download directory when collecting generated assets
- Capture final asset URLs or ids when the UI exposes them
