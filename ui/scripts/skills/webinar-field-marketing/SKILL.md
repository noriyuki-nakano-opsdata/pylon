# Webinar Field Marketing

Use this skill when the task is executing an event program with real follow-up requirements, not just writing a promo asset. It is optimized for registration flow, reminder cadence, attendance capture, and post-event lead handling.

## When to Use

- Planning a webinar or seminar runbook
- Designing invite and reminder cadence
- Preparing booth or field-event operating checklists
- Defining event lead capture and qualification flow
- Writing attendee and no-show follow-up plans
- Reviewing event outcomes and next-step routing

## Workflow

1. Define the event objective.
   State audience, topic, desired conversion, and hot-lead signal.
2. Build the runbook.
   Separate promotion, registration, reminders, live execution, and follow-up.
3. Define lead capture rules.
   Record what data is captured, who owns it next, and how hot vs nurture leads differ.
4. Split post-event actions.
   Create separate actions for attendees, no-shows, high-intent questions, and recycled leads.
5. Return the event operating pack.
   Include timeline, owner map, reminders, follow-up, and metrics.

## Output Shape

- Event brief: audience, promise, CTA, success metric
- Runbook: milestone, owner, date, checkpoint
- Lead capture plan: fields, tool, owner, hot-lead criteria
- Follow-up matrix: segment, message, timing, next owner
- Retro prompts: attendance, engagement, influenced pipeline, next improvements

## Heuristics

- Registration volume matters less than attendance plus follow-up quality.
- No-show flows should be designed before launch, not after the event.
- Booth staffing needs separate roles for attraction, conversation, and lead logging.
- The event is not complete until leads are routed and scheduled for follow-up.

## Anti-Patterns

- Sending the same follow-up to attendees and no-shows
- Treating badge scans as qualified leads by default
- Running an event without a hot-lead definition
- Measuring success only by registrations
